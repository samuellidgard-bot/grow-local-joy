# Xello Canva Build Pack

Created: 2026-05-08

Purpose: turn the template specs into repeatable Canva builds.

## Build These First

Create three Canva designs:

1. Prompt Giveaway
2. Myth Busting
3. Checklist/System

Each design should have:

- 6 slides.
- 1080x1350 size if possible.
- Square-safe margins so it can be adapted to 1080x1080.
- Xello logo bug.
- Slide number.
- Swipe cue on slide 1.
- Save/share CTA on final slide.

## Shared Brand Settings

Colours:

- Background: `#0B0D0F`
- Dark card: `#15181B`
- Border grey: `#2B3035`
- Text white: `#F4F6F8`
- Muted grey: `#A8AFB7`
- Xello green: `#9BE870`

Fonts:

- Use a clean bold sans-serif.
- Canva options: `Montserrat`, `Inter`, `DM Sans`, `League Spartan`.
- Use bold for hooks and medium/regular for small text.

Layout:

- Hook slide: 8-12 words maximum.
- Inner slides: one idea only.
- Final slide: one CTA only.
- Keep the logo small.
- Do not use tiny paragraphs.

## Canva Build Order

1. Build Checklist/System first.
2. Duplicate it and adapt to Myth Busting.
3. Duplicate again and adapt to Prompt Giveaway.
4. Save all three as reusable templates.
5. Export one test PNG and check readability on phone.

## Naming

Use these Canva names:

- `Xello - Checklist System Carousel`
- `Xello - Myth Busting Carousel`
- `Xello - Prompt Giveaway Carousel`

## Quality Check

Before using a template:

- Can slide 1 be read in one second?
- Does every slide have one main idea?
- Is the green accent guiding attention?
- Is the CTA large enough?
- Does it look like Xello, not a generic AI page?
