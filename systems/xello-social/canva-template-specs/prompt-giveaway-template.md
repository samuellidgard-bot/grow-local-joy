# Canva Template Spec: Prompt Giveaway

Created: 2026-05-08

Use for posts such as:

- "Steal this AI content prompt"
- "Use this prompt before posting"
- "Turn one service into 10 post ideas"

## Format

- Instagram carousel: 1080x1350 preferred, or 1080x1080 if keeping square.
- Reel cover: 1080x1920 variant.

## Visual System

- Background: dark graphite.
- Accent: Xello green.
- Main element: large prompt/code card.
- Small Xello logo bug bottom left.
- Copy icon or terminal cursor motif.

## Slide Structure

1. Big hook:
   `STEAL THIS AI CONTENT PROMPT ->`
2. Use case:
   `Use it when you don't know what to post.`
3. Prompt block:
   `Act as a content strategist for [BUSINESS TYPE]...`
4. Output fields:
   `Hook / On-screen text / Caption / CTA / Why it works`
5. Result:
   `Turn 1 service into 10 post ideas.`
6. CTA:
   `COMMENT PROMPT + SAVE FOR LATER.`

## Rules

- Keep prompt text readable.
- Do not fit too much code on one slide.
- Use green only for variables and CTA.
- Make the final slide feel like a useful action, not a sales pitch.
