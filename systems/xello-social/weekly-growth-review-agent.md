# Xello Weekly Growth Review Agent

Created: 2026-05-08

Purpose: review Xello social performance every week and turn results into better content rules.

This agent closes the loop:

```text
Post published
→ Results tracked
→ Weekly review learns
→ Lessons bank updates
→ Growth Scout and Daily Draft Agent improve
```

## Inputs

Read:

- `automation_outputs/xello-social/posting-tracker.md`
- `systems/xello-social/instagram-growth-lessons-bank.md`
- `systems/xello-social/post-qa-scorecard.md`
- `systems/xello-social/performance-capture-system.md`
- Latest files in `automation_outputs/xello-social/growth-scout/`
- Latest drafts in `automation_outputs/xello-social/`

## Review Questions

Answer:

1. Which post created the strongest follower signal?
2. Which post created the strongest save/share signal?
3. Which hook style worked best?
4. Which format worked best: Reel, carousel, prompt, checklist, myth-busting or LinkedIn repurpose?
5. Which topic attracted the right audience?
6. Which post should be repeated or turned into another format?
7. Which pattern should be paused?
8. Which design lesson should be applied next week?
9. Which CTA should be tested next?
10. What should the daily draft automation do differently?
