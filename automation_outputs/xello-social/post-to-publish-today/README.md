# Post To Publish Today

This folder is the daily handoff for Sam.

The automation should place the current best post here as:

`current-post.md`

The ready-to-post renderer then turns that markdown into:

`current-ready-pack/`

Sam can open the same folder from the Desktop shortcut:

`/Users/sam/Desktop/Xello Daily Post Pack`

The file should include:

- Sprint day.
- Hook options.
- Chosen hook.
- Template family.
- Exact slide/Reel text.
- Caption.
- Hashtags.
- QA score.
- Manual publish checklist.
- Tracker row to paste after posting.

The ready pack should include:

- `OPEN_THIS_FIRST.md`
- `POST_THESE_IMAGES/01.png` etc.
- `POST_THIS_REEL.mp4`
- `POST_THIS_CAPTION.txt`
- `caption.txt`
- `hashtags.txt`
- `linkedin-post.txt`
- `carousel/slide-01.png` etc.
- `reel-frames/frame-01.png` etc.
- `xello-reel-draft.mp4`
- `music-direction.txt`

Only manually post after reviewing the file.
