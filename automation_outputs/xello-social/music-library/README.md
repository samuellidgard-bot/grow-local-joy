# Xello Music Library

Add commercially safe music files here for automated Reel exports.

Recommended folders:

- `energetic-tech`
- `minimal-premium`
- `dark-ambient`
- `clean-corporate`
- `approved`

The daily ready-pack renderer creates an MP4 every time it runs. If a compatible track exists in `approved/`, it embeds that track. If not, it exports a silent MP4 so Sam can add music manually inside Instagram or TikTok.
