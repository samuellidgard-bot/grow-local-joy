# Approved Music Folder

Drop royalty-free tracks here when Xello has permission to use them.

Supported file types:

- `.mp3`
- `.m4a`
- `.wav`
- `.aac`

The daily ready-pack renderer uses the first compatible file it finds in this folder and embeds it into `xello-reel-draft.mp4`.

If this folder has no music, the renderer creates a silent MP4 so Sam can add music manually inside Instagram or TikTok.
