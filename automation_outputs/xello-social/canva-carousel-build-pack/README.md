# Xello Media Canva Carousel Build Pack

Date: 2026-05-12  
Purpose: create the next Xello Instagram carousel posts after the completed "What is Xello Media?" intro post.

## Important Canva Note

The Canva connector can see and edit Canva designs, but Canva template autofill is blocked unless the account has Canva Enterprise. Canva Pro is enough for you to use premium templates manually, but the API will not let Codex directly pull and autofill marketplace templates.

Best workflow from here:

1. In Canva, search one of the template terms below.
2. Pick a strong 5-7 page carousel template.
3. Share the Canva design link here.
4. Codex adapts the copy and Xello brand direction into that chosen template.

## Canva Template Search Terms

Use these in Canva templates:

- marketing tips carousel
- social media marketing carousel
- digital marketing carousel
- lead generation carousel
- business growth carousel
- black green carousel
- modern agency carousel
- tech marketing carousel
- checklist instagram carousel
- infographic instagram carousel
- bold typography carousel
- SaaS marketing carousel

Pick templates with:

- Big headline space.
- Strong contrast.
- 5-7 slides.
- Simple card sections.
- Room for icons.
- Not too much body text.
- Black/dark templates if possible.

Avoid:

- Beige lifestyle templates.
- Soft pastel coaching templates.
- Tiny text layouts.
- Overly busy collage templates.
- Templates built around stock people/office photos.
