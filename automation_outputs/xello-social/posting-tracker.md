# Xello Faceless Social Posting Tracker

Created: 2026-05-08

Use this tracker to move each automated draft from idea to manual post.

## Status Labels

- `Drafted`: automation created the post package.
- `Needs edit`: Sam should tweak wording/design before posting.
- `Ready`: safe to manually post.
- `Posted`: live on Instagram/TikTok/LinkedIn.
- `Measured`: performance logged after 24-72 hours.

## Posting Flow

1. Open the newest draft in `automation_outputs/xello-social/`.
2. Check the manual publish checklist.
3. Build the Reel/carousel in Canva, CapCut or Instagram drafts.
4. Post manually.
5. Add the live link and basic results here.
6. Feed the lesson back into the next batch.

## Daily Review Questions

- Did the hook make sense in one second?
- Did the post teach one clear thing?
- Was the save/share reason obvious?
- Did it attract business owners or random viewers?
- Did anyone ask a useful question?
- What should tomorrow's draft do differently?
