# Xello Social Weekly Reviews

Weekly reviews from `xello-social-weekly-growth-review` are saved here.

The weekly review should:

- Check posted results in `posting-tracker.md`.
- Identify winning hooks, formats and topics.
- Update `instagram-growth-lessons-bank.md` with reusable lessons.
- Tell Sam exactly what data is missing if posts have not been measured yet.

This is what turns the faceless account from a content calendar into a learning system.
